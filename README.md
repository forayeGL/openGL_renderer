# OpenGL_renderer
A simple rederer based on openGL, refering to LearnOpenGL.com.

## Content
Based on the OpenGL renderer implementation from learnOpenGL.com, most of the content on learnOpenGL has been implemented, including camera system, Phong lighting, model loading, depth and stencil testing, transparency and color blending, dual-pass post-processing, skyboxes, instanced rendering, gamma correction, normal mapping, parallax mapping, shadow mapping, PCF, PCSS, CSM cascaded shadows, point light shadows, MSAA, HDR, Bloom, and PBR. Based on this, demos such as a cartoon-style grass field and a NURBS surface renderer have also been completed.

## Prerequisites
CMake version 3.12 or higher, and a C++ compiler that supports C++11 or higher.

## Build
Just use cmake to build the project.
